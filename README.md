# Expense-Tracker

A simple and efficient command-line Expense Tracker built using:
Python
SQLite (SQL queries + views + triggers)
***********************************************************************
Features
Core Features
Add new expenses
List recent expenses
Search expenses by category, note, or date
Update existing expense
Delete expense
***********************************************************************
Reporting
View totals by category
SQL VIEW used for grouping & aggregation
Clean table formatting in CLI
***********************************************************************
SQL Features Used
SQL Views (v_totals_by_category)
SQL Triggers (optional audit history)
SQL constraints (CHECK, NOT NULL)
Parameterized queries (to prevent SQL injection)
***********************************************************************
